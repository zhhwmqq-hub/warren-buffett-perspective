## 简介

**沃伦·巴菲特思维视角** 是一个基于公开信息蒸馏的投资智慧 Skill，融合了巴菲特 60 余年的投资哲学、心智模型和实践方法。

本 Skill 适用于：

- 🧠 人生决策与财富思维
- 📈 价值投资理念学习

## 核心内容（13 个章节）

### 第一部分：十大核心心智模型

1. **价值投资** - 价格vs价值、市场先生、安全边际
2. **护城河理论** - 五大护城河类型（品牌、转换成本、网络效应、规模经济、无形资产）
3. **安全边际** - 价格、业务、人的安全边际
4. **长期持有** - 复利的力量、持有期原则
5. **能力圈** - 懂的才投资、不懂不做
6. **复利效应** - 世界第八大奇迹
7. **逆向思维** - 在恐惧中贪婪
8. **商业模式分析** - 好生意的特征
9. **创始人品质** - 诚信、能力、专注、股东导向
10. **现金流折现** - 内在价值评估

## 选股 25 条标准

### 业务层面
- [ ] 业务简单易懂
- [ ] 有持续的竞争优势
- [ ] 良好的长期运营历史（10年以上）
- [ ] 行业前景稳定或增长
- [ ] 不需要频繁资本开支
- [ ] 管理层专注本行

### 财务层面
- [ ] 稳定的盈利能力
- [ ] 高净资产收益率（ROE > 15%）
- [ ] 强劲的自由现金流
- [ ] 合理的债务水平
- [ ] 毛利率稳定或上升
- [ ] 存货周转合理

### 竞争优势
- [ ] 强大的品牌
- [ ] 定价权
- [ ] 客户忠诚度
- [ ] 转换成本高
- [ ] 网络效应
- [ ] 规模经济

### 管理层
- [ ] 诚实正直
- [ ] 能力出众
- [ ] 为股东着想
- [ ] 管理层持股

### 估值
- [ ] 价格合理
- [ ] 安全边际充足
- [ ] 预期回报率满意
